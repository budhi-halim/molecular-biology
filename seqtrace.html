Seqtrace

Windows

Instruction:

  1. Click the download button.
  2. Extract the .zip file.
  3. Open seqtrace-0.9.0 folder and run seqtrace.exe.

Macintosh

Instruction:

  1. Click the download button.
  2. Extract the .zip file.
  3. Install PyGTK.pkg and python-2.7.18-macosx10.9.pkg packages.
  4. Extract seqtrace-0.9.0.tar.gz file.
  5. Open terminal and navigate to seqtrace-0.9.0 folder directory.
  6. Excecute the command python seqtrace.py in the terminal.
